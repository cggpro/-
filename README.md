# 原神自动点击 - 使用 Golang 实现

## ✨ 功能：

1. 支持设置点击间隔时间和无动作超时自动关闭时间，单位为秒,不设置均为默认时间。
   
   示例1：命令行设置：
   ```
   click.exe 2 10
   表示点击间隔时间为2秒，超时自动关闭时间为10秒。
   ```
   示例2：双击后设置
   ![image](https://github.com/cggpro/GenshinClick/assets/120552503/73932f78-622f-44ee-845f-0aee6dcdd655)




3. 支持开启和关闭自动点击功能。

   - 按下 F9 键开启自动点击
   - 按下 F10 键关闭自动点击
